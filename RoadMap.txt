```
DevOps Roadmap for Freshers & Interns (2025)

1️⃣ Linux Basics
- Learn Linux command line (shell, file permissions, users/groups)
- Practice basic bash scripting
- Get comfortable with vim/nano editors
- Know basic networking commands (ping, netstat, curl, etc.)

2️⃣ Version Control (Git/GitHub)
- git init, commit, push, pull, branch
- resolving merge conflicts
- using GitHub workflows
- working with pull requests

3️⃣ Build Tools
- Maven (Java)
- Gradle (Java)
- npm/yarn (JavaScript)
- Understand build pipelines

4️⃣ CI/CD
- Learn Jenkins
- Learn GitHub Actions
- Understand CI/CD pipelines (build, test, deploy steps)

5️⃣ Containerization
- Learn Docker
  - images, containers, Dockerfile
  - docker-compose
- container registries (Docker Hub)

6️⃣ Orchestration
- Learn Kubernetes (K8s)
  - pods, deployments, services, configmaps
  - kubectl commands
- Use Minikube or KIND for local practice

7️⃣ Cloud Basics
- AWS (EC2, S3, IAM, basics of VPC)
- OR Azure OR GCP
- practice deploying simple apps to cloud

8️⃣ Infrastructure as Code (IaC)
- Terraform basics
- how to provision infrastructure

9️⃣ Monitoring & Logging
- Prometheus / Grafana
- ELK Stack (Elasticsearch, Logstash, Kibana)
- AWS CloudWatch

🔟 Soft Skills
- problem solving
- team communication (slack, jira, confluence)
- documentation skills

BONUS:
- Learn basic networking concepts (TCP/IP, DNS, load balancing)
- Learn a scripting/programming language (Python is best for automation)
- Contribute to open-source DevOps projects on GitHub

Tip:
Keep building small projects and share them on GitHub to strengthen your profile.
```

✅ તમારે જો હોમવર્ક તરીકે **weekly plan** પણ જોઈએ, તો કહેજો — હું તમને schedule બનાવી આપીશ, week 1, week 2 પ્રમાણે. 🚀
